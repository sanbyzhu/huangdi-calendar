# 贡献指南

感谢你愿意帮助古历法变得更准确、更好用。

## 如何参与

- 提交 Issue：反馈 Bug、资料错误、历法校验结果或体验建议。
- 提交 Pull Request：修复问题、补充资料、优化界面或完善文档。
- 校对资料：请尽量附上出处、日期范围和验证方式。

## 本地开发

```bash
npm install
npm run dev
```

提交前建议运行：

```bash
npm run lint
npm run build
```

## 提交建议

- 让改动保持聚焦，一次 PR 解决一个清晰问题。
- 涉及历法算法时，请补充说明基准日期、推导依据和测试样例。
- 涉及 UI 时，请说明桌面端和移动端的主要验证结果。
- 不要提交 `dist/`、`release/`、安装包、日志、缓存或本地私有配置。
