# 《古历法》应用详细介绍 (V0.10.07)

## 1. 项目愿景
《古历法》不仅是一款工具类应用，更是一次横跨千年的文化溯源之旅。它的诞生旨在将晦涩难懂的华夏古天文知识，通过现代数字化手段与极致的视觉设计，转化为触手可及的交互体验。我们希望每一个用户在拨动时间轮盘时，都能感受到祖先仰望星空时的那份浩瀚与激昂。

---

## 2. 核心特色

### 2.1 多维历法闭环系统
应用集成了四套各具特色的历法体系，并实现了毫秒级的实时同步：
- **黄帝纪元**：溯源华夏文明始祖的时间坐标。
- **圣太一天历**：不同于传统的阴历，这是基于地球公转角度、以回归年为绝对锚点的十月制太阳历模型。
- **夏历（农历）**：承载民俗、节气与阴阳平衡的民族瑰宝。
- **玛雅历法**：引入遥远中美洲文明的“天玺”与“太阳历”，展现人类文明对时间的共同探索。

### 2.2 极致天文视觉工程
- **星系动力学模拟**：采用真实的轨道参数比例（半对数处理），模拟六大行星的运行状态。
- **大数流光系统**：背景中飘动的“由、秭、穰、沟、涧...”等大数单位，将中国传统的无限量级概念具象化。
- **北斗九星坐标**：背景融入北斗七星及隐藏的“辅、弼”二星，构成完整的九星格局，指引视觉方位。

### 2.3 互动黑科技：天意合一
- **七星连珠**：通过复杂的弹簧物理算法，模拟百年一遇的行星大对齐。在一片璀璨的金色流光中，用户可以亲手开启属于自己的“天降奇观”。
- **全息推演中心**：支持跨纪元的时间穿梭，用户可以通过旋转拨盘，对比不同纪元在宇宙尺度下的变动规律。

---

## 3. 技术方案
- **引擎**：基于 **React 19** 与 **Vite** 构建，确保极其流畅的 UI 交互。
- **动画**：深度利用 **Framer Motion** 实现物理级别的平滑过渡与高帧率动画。
- **图标**：采用 **Lucide** 系列图标，兼具极简主义与辨识度。
- **跨平台**：通过 **Capacitor** 技术栈，实现 Web 项目向原生 Android App 的完美转化。
